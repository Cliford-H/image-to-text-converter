
# Image to Text Converter (OCR)

A simple Python project that extracts text from images using Optical Character Recognition (OCR).

## Tech Stack
- Python
- Pytesseract
- Pillow

## How to Run

1. Install dependencies  
   pip install -r requirements.txt

2. Run the project  
   python app.py

## Use Cases
- Document digitization
- Extracting text from scanned images
- Automation of data entry
